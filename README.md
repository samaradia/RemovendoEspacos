# Estudos de Strings em Java

Este repositório contém exercícios simples para praticar manipulação de Strings em Java.

## Conteúdo

- `trim()` - atual.
- `length()`
- `substring()`
- `toUpperCase()`
- `toLowerCase()`
- Outros métodos de String

## Objetivo

Aprender e praticar os principais métodos da classe `String` através de exemplos simples e exercícios.

## Tecnologias

- Java
- IntelliJ IDEA (ou outra IDE)
